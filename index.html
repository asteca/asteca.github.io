About

The Automated Stellar Cluster Analysis (ASteCA) tool is designed to fully automatize the usual tests applied on star clusters (SC) in order to determine their characteristics: center, radius, stars' membership probabilities and associated intrinsic/extrinsic parameters: metallicity, age, reddening, distance and total mass.

ASteCA is developed entirely in Python and it's distributed as open source code under a GPLv3 license.
